# 心理健康评估量表

一个伪装成正经心理健康测试的恐怖互动体验。50 道渐进式问题，13 种惊吓效果，在你放松警惕时突然袭击。

## 玩法

看起来是一份普通的 SCL-90 心理健康问卷。但某些问题的选项会触发意外事件——页面闪烁、诡异音效、弹幕覆盖、鬼脸浮现……测试越往后越不对劲，而你已经开始怀疑：这个测试真的只是在评估心理健康吗？

## 技术

纯前端 HTML/CSS/JS，无需构建工具。打开 `index.html` 即可运行。

```
├── index.html   # 主页面（50 题 + 13 种惊吓事件）
├── audio/       # 音效文件（心跳、环境音、敲门声、笑声等）
└── images/      # 图片素材（鬼脸、诡异医生）
```

## 本地运行

```bash
npx serve -l 5173 -s .
```

然后打开 http://localhost:5173

## 免责声明

本测试纯属娱乐，不具备任何心理评估功能。如果你在测试过程中感到不适，请关闭页面——它可能比你更了解你自己。
